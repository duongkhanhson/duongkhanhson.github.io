# Phone Camera Face Recognition UI

A clean GitHub Pages landing page for the phone camera face recognition project.

## What this project does

This project demonstrates a Python desktop UI that connects to a mobile phone camera stream and performs real-time face detection.

- Live video stream from a phone camera via HTTP/MJPEG.
- Face detection using OpenCV Haar cascades.
- Snapshot capture from the live stream.
- Simple desktop interface built with Tkinter.

## Why it is useful

This application is ideal for rapid prototyping of vision applications and for demonstrating how to combine mobile camera streaming with Python computer vision.

## Quick start

```bash
pip install -r phone-camera-face-recognition/requirements.txt
python phone-camera-face-recognition/phone_cam_face_gui.py
```
