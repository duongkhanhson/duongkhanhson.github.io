## 📌 Introduction
The **PID Motor Serial Control** project bridges the gap between complex control theory algorithms and visual data representation. It provides a real-time graphical interface to tune Proportional, Integral, and Derivative (PID) parameters for an Arduino-controlled DC Motor.

## ✨ Features
- **Real-Time Tuning:** Dynamically adjust Kp, Ki, and Kd values via sliders, instantly transmitting them over a UART Serial connection to the embedded board.
- **Data Plotting:** Visualizes the dynamic response of the motor (Set-point vs. Process Value) to easily identify overshoot, steady-state error, and settling time.
- **Mechatronics Core:** Designed for control systems education and prototyping automated joint mechanisms.
