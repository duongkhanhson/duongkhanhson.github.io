## 📌 Introduction
This system extends simple computer vision into behavioral monitoring by precisely localizing both human faces and specific eye coordinates. 

## ✨ Features
- **Hierarchical Detection:** First localizes the Region of Interest (ROI) containing a face, then processes bounding boxes strictly within that ROI to isolate eyes.
- **Real-Time Processing:** Optimized detection loops ensuring high framerates using basic webcam feeds.
- **Foundation for Drowsiness Detection:** Acts as the primary architectural stepping stone for building Automated Driver Fatigue or User Attention monitoring systems in academic research.
