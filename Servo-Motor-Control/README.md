## 📌 Introduction
This project demonstrates an architecture to strictly command and orchestrate a multi-DOF (Degrees of Freedom) Servo Motor array. Utilizing Python for high-level logic and an embedded microcontroller for low-level PWM signal generation.

## ✨ Features
- **Multi-Axis Control:** Independently control the angular displacement (0° - 180°) of multiple servo motors.
- **Serial Protocol:** Implements a robust packet-based serial communication standard between the Python Host and Arduino Slave.
- **GUI Operations:** Simple slider-based interactions map mathematically to the PWM duty cycles driving the mechanical linkages.
