# Projects

This page highlights the key projects in the repository, presented as a professional portfolio section.

## Featured Project

### Phone Camera Face Recognition UI
- Folder: `phone-camera-face-recognition`
- Description: Desktop Python app that connects to a phone camera MJPEG stream and performs real-time face detection with OpenCV and Tkinter.
- Link: `/phone-camera-face-recognition/`
